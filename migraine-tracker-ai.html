Migraine-Tracker-AI-Privacy-Policy Privacy Policy This Privacy Policy applies to the Migraine Tracker AI – Headache Diary & Trigger Analyzer mobile application (hereinafter referred to as the “Application”) developed and published by Sergey Medvedev (hereinafter referred to as the “Developer”, “Service Provider”, “we”, “us”, or “our”). The Application is distributed through Google Play as a Freemium service and is provided “AS IS” without warranties of any kind. Effective Date: March 12, 2026 Information We Collect and How We Use It The Application is a personal migraine and headache tracking tool designed to help users monitor symptoms and identify possible triggers. Users may voluntarily enter health-related information into the Application, including but not limited to: Date and time of headache episodes Pain level (0–10) Duration of headaches Location of pain Associated symptoms (e.g., nausea, sensitivity to light or sound, aura) Possible triggers (e.g., stress, sleep patterns, diet, weather) Personal notes This information (User Health Data) is stored locally on the user’s device only using an Android Room database. The Developer does not collect, store, transmit, or access this User Health Data on external servers. Artificial Intelligence Features The Application may include optional AI-powered analysis features designed to help users identify patterns or potential migraine triggers. When a user voluntarily initiates an AI analysis request: Only the minimum necessary anonymized excerpts of migraine logs may be temporarily transmitted to an AI service provider (such as Google AI services like Gemini). The transmitted data does not include personal identifiers. The Developer does not store or retain this data after the analysis is completed. AI providers process data according to their own privacy policies. Automatically Collected Information The Application automatically collects limited non-personally identifiable technical information to improve performance and reliability. This may include: Device manufacturer and model Operating system version Application version and build number Crash reports and diagnostic information General app usage events (e.g., app opens, screen views) This information is collected through services provided by Google, including: Firebase Analytics Firebase Crashlytics Firebase Remote Config This data helps us understand how the Application performs and improve user experience. Advertising The Application may display advertisements using Google AdMob. AdMob may collect limited information such as: Advertising Identifier (Advertising ID) Approximate location derived from IP address Device information Ad interaction events This information is used to deliver relevant advertisements and measure ad performance. You can disable personalized advertising in your device settings. App Permissions The Application may request the following permissions: Internet Access – required for: Advertising (Google AdMob) Analytics (Firebase Analytics) Crash reporting (Firebase Crashlytics) Optional AI analysis features These permissions are used only for the purposes described in this Privacy Policy. Third-Party Services The Application may use the following third-party services: Google AdMob (Advertising) Firebase Analytics (Usage analytics) Firebase Crashlytics (Crash reporting) Firebase Remote Config (Feature configuration) Google Play Billing (Subscriptions and in-app purchases) These services may collect and process data in accordance with their own privacy policies: https://policies.google.com/privacy Data Retention User Health Data entered in the Application remains stored locally on the user’s device until: the user deletes it manually the user clears application data the user uninstalls the Application Automatically collected analytics or advertising data may be retained by Google services according to their own retention policies. Opt-Out and Data Deletion Users can stop all data collection by: Uninstalling the Application Resetting the Advertising ID in device settings Disabling personalized ads in Android settings If you have questions regarding data deletion requests related to third-party services, you may contact us.
